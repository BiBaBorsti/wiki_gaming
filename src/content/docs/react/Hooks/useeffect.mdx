# Der React Hook `useEffect`

Der React Hook `useEffect` ist eine Möglichkeit, in React-Funktionskomponenten side effects (Nebeneffekte) zu verwalten. Side effects sind Aktionen, die außerhalb des normalen Flusses der Komponentenrendering-Aktualisierung stattfinden, wie beispielsweise das Abrufen von Daten von einem Server, das Abonnieren von Ereignissen oder das Ändern des DOM außerhalb des Renders.

Die grundlegende Syntax von `useEffect` lautet:

```jsx
useEffect(() => {
  // Code für den Nebeneffekt
}, [abhängigkeiten]);
```

- Der erste Parameter ist eine Funktion, die den Code für den auszuführenden Nebeneffekt enthält.
- Der optionale zweite Parameter ist ein Array von Abhängigkeiten. Wenn sich eine Abhängigkeit ändert, wird der Nebeneffekt erneut ausgelöst. Wenn das Abhängigkeiten-Array leer ist, wird der Nebeneffekt nach jeder Aktualisierung ausgelöst.

## Anwendungsbeispiele für `useEffect`

### 1. Datenabruf von einem Server

```jsx
import React, { useState, useEffect } from 'react';

function DataFetchingComponent() {
  const [data, setData] = useState([]);

  useEffect(() => {
    fetch('https://api.example.com/data')
      .then(response => response.json())
      .then(data => setData(data));
  }, []);

  return (
    <div>
      {data.map(item => (
        <p key={item.id}>{item.name}</p>
      ))}
    </div>
  );
}
```

### 2. DOM-Manipulation nach dem Rendern

```jsx
import React, { useState, useEffect } from 'react';

function DocumentTitleUpdater() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `Count: ${count}`;
  }, [count]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
```

### 3. Ereignisabonnement und -aufräumarbeiten

```jsx
import React, { useState, useEffect } from 'react';

function EventListenerComponent() {
  const [windowWidth, setWindowWidth] = useState(window.innerWidth);

  const handleResize = () => {
    setWindowWidth(window.innerWidth);
  };

  useEffect(() => {
    window.addEventListener('resize', handleResize);

    return () => {
      window.removeEventListener('resize', handleResize);
    };
  }, []);

  return <p>Window width: {windowWidth}</p>;
}
```

Der `useEffect`-Hook ist äußerst nützlich, um Nebeneffekte in React-Funktionskomponenten zu verwalten und sicherzustellen, dass diese Effekte ordnungsgemäß synchronisiert und aufgeräumt werden, um potenzielle Lecks und unerwartetes Verhalten zu vermeiden.
